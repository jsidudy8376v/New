## ⚙️ How to Deploy

1. Clone this repository.
2. Set your `config.py` with API_ID, API_HASH, BOT_TOKEN & other required settings.
3. Install dependencies:
    ```
    pip install -r requirements.txt
    ```
4. Run the bot:
    ```
    python3 main.py
    ```
